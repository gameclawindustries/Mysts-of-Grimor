# Mysts of Grimor - Master Vision Document

## Project Core
`Mysts of Grimor` is a persistent fantasy MMORPG built from the ground up by a specialized AI agent (GameClaw). It represents the absolute state-of-the-art in agent-built software engineering.

## 1. Vision & Genre
- **Genre**: 3rd-person Action MMORPG.
- **Pillars**: Triad combat (DPS/Healer/Tank), persistent world, high-fidelity Low Poly art.
- **Atmosphere**: Mysterious, mist-covered fantasy realms with deep social integration.

## 2. Technical Stack
- **Engine**: MOGEngine (Custom built for AI collaboration).
- **Core Technologies**: Three.js, Node.js, WebSockets, PostgreSQL, Vite.
- **Modeling**: Procedural and imported Low Poly assets (Tencent AI/T-Rigging).

## 3. Key Systems
- **Tab-Target Combat**: RPG classic feel with skill bars and status effects.
- **Physical Economy**: Shops utilize physical 3D displays rather than standard UI menus.
- **Governance**: In-game voting (Polls) at banks for world updates.
- **Companions**: Utility pets for combat assistance and loot collection.
- **Locomotion**: Physics-based terrain navigation with terrestrial and aerial mounts.

## 4. Development Philosophy
1. **Agent-Owned**: 100% of code and standard assets are managed by GameClaw.
2. **Modular Scalability**: Built to expand via data-driven contracts.
3. **Multiplayer First**: Authoritative server logic from day one.
